# mervetekgurler.github.io

Personal and academic website of Merve Tekgürler, available at [mervetekgurler.com](https://mervetekgurler.com).

## Stack

- [Jekyll](https://jekyllrb.com/) with the [Minima](https://github.com/jekyll/minima) theme
- Hosted on [GitHub Pages](https://pages.github.com/)
- Custom CSS in `assets/css/custom.css` (Roboto font, warm off-white palette)

## Structure

- `index.md` — homepage / bio
- `_pages/publications.md` — publications list
- `_posts/` — blog posts
- `blog/index.html` — paginated blog index
- `materials/` — images and files referenced by posts
- `assets/css/custom.css` — all theme overrides
- `_includes/head-custom.html` — favicon and CSS injection
